# Contributing to Trading System

## Code of Conduct

Be respectful and professional in all interactions.

## Getting Started

1. Fork the repository
2. Clone your fork: git clone https://github.com/yourusername/trading-system.git
3. Create a feature branch: git checkout -b feature/your-feature
4. Install dependencies: pip install -r requirements.txt
5. Make your changes
6. Add tests for new functionality
7. Run tests: pytest tests/ -v
8. Commit changes: git commit -am 'Add feature'
9. Push to branch: git push origin feature/your-feature
10. Create a Pull Request

## Code Standards

### Python Style Guide
- Follow PEP 8
- Use flake8 for linting
- Use type hints where possible
- Maximum line length: 120 characters

### Testing
- Write unit tests for new features
- Maintain or improve code coverage
- Run full test suite before submitting PR

## Commit Messages

Use clear, descriptive commit messages
